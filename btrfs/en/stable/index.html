Welcome to BTRFS documentation!
欢迎阅读 BTRFS 文档！

BTRFS is a modern copy on write (COW) filesystem for Linux aimed at implementing advanced features while also focusing on fault tolerance, repair and easy administration. You can read more about the features in the introduction or choose from the pages below. Documentation for for command line tools btrfs(8), mkfs.btrfs(8) and others is in the manual pages.
BTRFS 是一个现代的写时复制（COW）文件系统，专为 Linux 设计，旨在实现高级功能的同时也专注于容错性、修复和简易管理。您可以在介绍中了解更多功能，或从下面的页面中进行选择。有关命令行工具 btrfs(8)、mkfs.btrfs(8) 和其他工具的文档，请参阅手册页。
