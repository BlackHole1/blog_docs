FileSearch element (Util extension)
文件搜索元素（Util 扩展）

Describes a file search. 描述文件搜索。

Parents​ 父母

Bundle, Fragment 捆绑包，片段

Attributes​ 属性

After (String) : Id of the search that this one should come after.
之后 (字符串): 此搜索应在其之后进行的搜索的 ID。

Condition (String) : Condition for evaluating the search. If this evaluates to false, the search is not executed at all.
条件 (字符串): 用于评估搜索的条件。如果此条件评估为假，则根本不执行搜索。

DisableFileRedirection (wxs:YesNoTypeUnion) : When set to "yes" and the running bundle is 32-bit, Wow64DisableWow64FsRedirection is called before starting the search.
禁用文件重定向（wxs:YesNoTypeUnion）：当设置为“是”且运行的包是 32 位时，在开始搜索之前调用 Wow64DisableWow64FsRedirection。

Id (String) : Id of the search for ordering and dependency.
ID (字符串): 用于排序和依赖的搜索 ID。

Path (String) : File path to search for.
路径（字符串）：要搜索的文件路径。

Result (enumeration) : Rather than saving the matching file path into the variable, a FileSearch can save an attribute of the matching file instead. This attribute's value must be one of the following:
结果（枚举）：文件搜索可以将匹配的文件路径保存到变量中，也可以保存匹配文件的属性。此属性的值必须是以下之一：

  • exists: Saves true if a matching file is found; false otherwise.
    存在：如果找到匹配的文件，则保存为 true；否则为 false。
  • version: Saves the version information for files that have it (.exe, .dll); zero-version (0.0.0.0) otherwise.
    版本：保存有版本信息的文件的版本信息（.exe，.dll）；否则为零版本（0.0.0.0）。

Variable (String, required) : Name of the variable in which to place the result of the search.
变量（字符串，必填）：放置搜索结果的变量名称。
