Certificate element (Iis extension)
证书元素（Iis 扩展）

Used to install and uninstall certificates.
用于安装和卸载证书。

Parents​ 父母

Component 组件

Attributes​ 属性

BinaryRef (String) : Reference to a Binary element that will store the certificate as a stream inside the package. This attribute cannot be specified with the CertificatePath attribute.
BinaryRef（字符串）：引用一个二进制元素，该元素将在包内以流的形式存储证书。此属性不能与 CertificatePath 属性一起指定。

CertificatePath (String) : If the Request attribute is "no" then this attribute is the path to the certificate file outside of the package. If the Request attribute is "yes" then this atribute is the certificate authority to request the certificate from. This attribute may be set via a formatted Property (e.g. [MyProperty]).
CertificatePath（字符串）：如果 Request 属性为“否”，则此属性是包外证书文件的路径。如果 Request 属性为“是”，则此属性是请求证书的证书颁发机构。此属性可以通过格式化的属性设置（例如，[MyProperty]）。

Id (String) : Unique identifier for this certificate in the installation package. If the Id is not specified, it will be generated.
Id（字符串）：安装包中此证书的唯一标识符。如果未指定 Id，将自动生成。

Name (String, required) : Name of the certificate that will be installed or uninstalled in the specified store. This attribute may be set via a formatted Property (e.g. [MyProperty]).
名称（字符串，必填）：将在指定存储中安装或卸载的证书名称。此属性可以通过格式化属性设置（例如，[MyProperty]）。

PFXPassword (String) : If the Binary stream or path to the file outside of the package is a password protected PFX file, the password for that PFX must be specified here. This attribute may be set via a formatted Property (e.g. [MyProperty]).
PFX 密码（字符串）：如果二进制流或包外文件路径是受密码保护的 PFX 文件，则必须在此处指定该 PFX 的密码。此属性可以通过格式化属性设置（例如，[MyProperty]）。

Request (wxs:YesNoTypeUnion) : This attribute controls whether the CertificatePath attribute is a path to a certificate file (Request='no') or the certificate authority to request the certificate from (Request='yes').
请求 (wxs:YesNoTypeUnion) : 此属性控制 CertificatePath 属性是证书文件的路径 (Request='no') 还是请求证书的证书颁发机构 (Request='yes')。

StoreLocation (enumeration, required) : This attribute's value must be one of the following:
存储位置 (枚举，必需) : 此属性的值必须是以下之一：

  • currentUser 当前用户
  • localMachine 本地机器

StoreName (enumeration, required) : This attribute's value must be one of the following:
商店名称（枚举，必需）：此属性的值必须是以下之一：

  • ca: Contains the certificates of certificate authorities that the user trusts to issue certificates to others. Certificates in these stores are normally supplied with the operating system or by the user's network administrator.
    ca：包含用户信任的证书颁发机构的证书。这些存储中的证书通常由操作系统或用户的网络管理员提供。
  • my: Use the "personal" value instead.
    my: 请使用“personal”值代替。
  • personal: Contains personal certificates. These certificates will usually have an associated private key. This store is often referred to as the "MY" certificate store.
    personal: 包含个人证书。这些证书通常会有一个关联的私钥。这个存储通常被称为“MY”证书存储。
  • request
  • root: Contains the certificates of certificate authorities that the user trusts to issue certificates to others. Certificates in these stores are normally supplied with the operating system or by the user's network administrator. Certificates in this store are typically self-signed.
    root：包含用户信任的证书颁发机构的证书，这些机构向其他人颁发证书。此存储中的证书通常由操作系统或用户的网络管理员提供。此存储中的证书通常是自签名的。
  • otherPeople: Contains the certificates of those that the user normally sends enveloped messages to or receives signed messages from. See MSDN documentation for more information.
    otherPeople：包含用户通常发送加密消息或接收签名消息的人的证书。有关更多信息，请参阅 MSDN 文档。
  • trustedPeople: Contains the certificates of those directly trusted people and resources. See MSDN documentation for more information.
    trustedPeople：包含直接信任的人员和资源的证书。有关更多信息，请参阅 MSDN 文档。
  • trustedPublisher: Contains the certificates of those publishers who are trusted. See MSDN documentation for more information.
    trustedPublisher: 包含受信任发布者的证书。有关更多信息，请参阅 MSDN 文档。

Vital (wxs:YesNoTypeUnion) : Indicates whether to fail the install if the certificate fails to install. The default is "yes".
重要 (wxs:YesNoTypeUnion) : 指示如果证书安装失败是否安装失败。默认值为“是”。
