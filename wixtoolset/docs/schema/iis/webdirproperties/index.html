WebDirProperties element (Iis extension)
WebDirProperties 元素（Iis 扩展）

WebDirProperties used by one or more WebSites. Lists properties common to IIS web sites and vroots. Corresponding properties can be viewed through the IIS Manager snap-in. One property entry can be reused by multiple sites or vroots using the Id field as a reference, using WebVirtualDir.DirProperties, WebSite.DirProperties, or WebDir.DirProperties.
WebDirProperties 被一个或多个网站使用。列出 IIS 网站和虚拟目录的常见属性。可以通过 IIS 管理器管理单元查看相应的属性。一个属性条目可以被多个站点或虚拟目录重复使用，使用 Id 字段作为引用，使用 WebVirtualDir.DirProperties、WebSite.DirProperties 或 WebDir.DirProperties。

Attributes​ 属性

AccessSSL (wxs:YesNoTypeUnion) : A value of true indicates that file access requires SSL file permission processing, with or without a client certificate. This corresponds to AccessSSL flag for AccessSSLFlags IIS metabase property.
AccessSSL (wxs:YesNoTypeUnion) : 值为 true 表示文件访问需要 SSL 文件权限处理，无论是否有客户端证书。这对应于 AccessSSLFlags IIS 元数据库属性的 AccessSSL 标志。

AccessSSL128 (wxs:YesNoTypeUnion) : A value of true indicates that file access requires SSL file permission processing with a minimum key size of 128 bits, with or without a client certificate. This corresponds to AccessSSL128 flag for AccessSSLFlags IIS metabase property.
AccessSSL128 (wxs:YesNoTypeUnion) : 值为 true 表示文件访问需要 SSL 文件权限处理，最小密钥大小为 128 位，无论是否有客户端证书。这对应于 AccessSSLFlags IIS 元数据库属性的 AccessSSL128 标志。

AccessSSLMapCert (wxs:YesNoTypeUnion) : This corresponds to AccessSSLMapCert flag for AccessSSLFlags IIS metabase property.
AccessSSLMapCert (wxs:YesNoTypeUnion) : 这对应于 AccessSSLFlags IIS 元数据库属性的 AccessSSLMapCert 标志。

AccessSSLNegotiateCert (wxs:YesNoTypeUnion) : This corresponds to AccessSSLNegotiateCert flag for AccessSSLFlags IIS metabase property.
AccessSSLNegotiateCert (wxs:YesNoTypeUnion) : 这对应于 AccessSSLFlags IIS 元数据库属性的 AccessSSLNegotiateCert 标志。

AccessSSLRequireCert (wxs:YesNoTypeUnion) : This corresponds to AccessSSLRequireCert flag for AccessSSLFlags IIS metabase property.
AccessSSLRequireCert (wxs:YesNoTypeUnion) : 这对应于 AccessSSLFlags IIS 元数据库属性的 AccessSSLRequireCert 标志。

AnonymousAccess (wxs:YesNoTypeUnion) : Sets the Enable Anonymous Access checkbox, which maps anonymous users to a Windows user account. When setting this to 'yes' you should also provide the user account using the AnonymousUser attribute, and determine what setting to use for the IIsControlledPassword attribute. Defaults to 'no.'
AnonymousAccess (wxs:YesNoTypeUnion) : 设置启用匿名访问复选框，将匿名用户映射到 Windows 用户帐户。将此设置为“是”时，还应使用 AnonymousUser 属性提供用户帐户，并确定 IIsControlledPassword 属性的设置。默认设置为“否”。

AnonymousUser (String) : Reference to the Id attribute on the User element to be used as the anonymous user for the directory. See the User element for more information.
AnonymousUser（字符串）：引用用户元素上的 Id 属性，用作目录的匿名用户。有关更多信息，请参见用户元素。

AspDetailedError (wxs:YesNoTypeUnion) : Sets the option for whether to send detailed ASP errors back to the client on script error. Default is 'no.'
AspDetailedError（wxs:YesNoTypeUnion）：设置是否在脚本错误时将详细的 ASP 错误发送回客户端的选项。默认值为“否”。

AuthenticationProviders (String) : Comma delimited list, in order of precedence, of Windows authentication providers that IIS will attempt to use: NTLM, Kerberos, Negotiate, and others.
AuthenticationProviders（字符串）：IIS 将尝试使用的 Windows 身份验证提供程序的逗号分隔列表，按优先顺序排列：NTLM、Kerberos、Negotiate 等。

BasicAuthentication (wxs:YesNoTypeUnion) : Sets the Basic Authentication option, which allows clients to provide credentials in plaintext over the wire. Defaults to 'no.'
BasicAuthentication (wxs:YesNoTypeUnion) : 设置基本身份验证选项，允许客户端通过网络以明文形式提供凭据。默认值为“否”。

CacheControlCustom (String) : Custom HTTP 1.1 cache control directives.
CacheControlCustom (String) : 自定义 HTTP 1.1 缓存控制指令。

CacheControlMaxAge (xs:nonNegativeInteger) : Integer value specifying the cache control maximum age value.
CacheControlMaxAge (xs:nonNegativeInteger) : 指定缓存控制最大年龄值的整数值。

ClearCustomError (wxs:YesNoTypeUnion) : Specifies whether IIs will return custom errors for this directory.
ClearCustomError (wxs:YesNoTypeUnion) : 指定 IIs 是否为此目录返回自定义错误。

DefaultDocuments (String) : The list of default documents to set for this web directory, in comma-delimited format.
DefaultDocuments (String) : 要为此 Web 目录设置的默认文档列表，以逗号分隔的格式。

DigestAuthentication (wxs:YesNoTypeUnion) : Sets the Digest Authentication option, which allows using digest authentication with domain user accounts. Defaults to 'no.'
DigestAuthentication (wxs:YesNoTypeUnion) : 设置摘要身份验证选项，允许使用域用户帐户进行摘要身份验证。默认值为“否”。

Execute (wxs:YesNoTypeUnion) :
执行 (wxs:YesNoTypeUnion) :

HttpExpires (String) : Value to set the HttpExpires attribute to for a Web Dir in the metabase.
HttpExpires (字符串) ：在元数据库中为 Web 目录设置 HttpExpires 属性的值。

Id (String) : Identifier for the WebDirProperties. The Id is required when the parent is a Fragment, Module, or Package. Otherwise, Id is optional and if not specified, it will be generated.
Id (字符串) ：WebDirProperties 的标识符。当父级是 Fragment、Module 或 Package 时，Id 是必需的。否则，Id 是可选的，如果未指定，将会生成。

IIsControlledPassword (wxs:YesNoTypeUnion) : Sets whether IIS should control the password used for the Windows account specified in the AnonymousUser attribute. Defaults to 'no.'
IIsControlledPassword (wxs:YesNoTypeUnion) : 设置是否应由 IIS 控制在 AnonymousUser 属性中指定的 Windows 帐户使用的密码。默认值为“否”。

Index (wxs:YesNoTypeUnion) : Sets the Index Resource option, which specifies whether this web directory should be indexed. Defaults to 'no.'
Index (wxs:YesNoTypeUnion) : 设置索引资源选项，指定是否应对该 Web 目录进行索引。默认值为“否”。

LogVisits (wxs:YesNoTypeUnion) : Sets whether visits to this site should be logged. Defaults to 'no.'
LogVisits (wxs:YesNoTypeUnion) : 设置是否应记录对该站点的访问。默认值为“否”。

PassportAuthentication (wxs:YesNoTypeUnion) : Sets the Passport Authentication option, which allows clients to provide credentials via a .Net Passport account. Defaults to 'no.'
PassportAuthentication (wxs:YesNoTypeUnion) : 设置护照认证选项，允许客户端通过.Net 护照帐户提供凭据。默认值为“否”。
