Font element (Thmutil extension)
字体元素（Thmutil 扩展）

Defines a font including the size and color.
定义字体，包括大小和颜色。

Parents​ 父母

Theme 主题

Attributes​ 属性

Background (FontColorType) : A system color id or a hexadecimal value representing BGR background color of the font. "ffffff" is white, "ff0000" is pure blue, "00ff00" is pure green, "0000ff" is pure red, and "000000" is black. If this attribute is absent the background will be transparent. Supported system color ids are: btnface, btntext, graytext, highlight, highlighttext, hotlight, window, and windowtext.
背景（FontColorType）：系统颜色 ID 或表示字体背景颜色的 BGR 十六进制值。"ffffff"是白色，"ff0000"是纯蓝色，"00ff00"是纯绿色，"0000ff"是纯红色，"000000"是黑色。如果此属性不存在，背景将是透明的。支持的系统颜色 ID 有：btnface、btntext、graytext、highlight、highlighttext、hotlight、window 和 windowtext。

Foreground (FontColorType) : A system color id or a hexadecimal value representing BGR foreground color of the font. "ffffff" is white, "ff0000" is pure blue, "00ff00" is pure green, "0000ff" is pure red, and "000000" is black. If this attribute is absent the foreground will be transparent. Supported system color ids are: btnface, btntext, graytext, highlight, highlighttext, hotlight, window, and windowtext.
前景（FontColorType）：系统颜色 ID 或表示字体前景颜色的 BGR 十六进制值。"ffffff"是白色，"ff0000"是纯蓝色，"00ff00"是纯绿色，"0000ff"是纯红色，"000000"是黑色。如果此属性不存在，前景将是透明的。支持的系统颜色 ID 有：btnface、btntext、graytext、highlight、highlighttext、hotlight、window 和 windowtext。

Height (xs:int, required) : Font size. Use negative numbers to specify the font in pixels.
高度（xs:int，必需）：字体大小。使用负数指定字体的像素大小。

Id (String, required) : Identifier for the font.
Id（字符串，必填）：字体的标识符。

Underline (enumeration) : Specifies whether the font is underlined. This attribute's value must be one of the following:
下划线（枚举）：指定字体是否带有下划线。此属性的值必须是以下之一：

  • no
  • yes

Weight (xs:nonNegativeInteger) : Font weight.
粗细（xs:nonNegativeInteger）：字体粗细。
