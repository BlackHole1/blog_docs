File Element 文件元素

Description 描述

文件表的文件规范，必须是组件的子节点。

Choice of elements (min: 0, max: unbounded)
元素选择（最小值：0，最大值：无限）
  • AppId (min: 0, max: unbounded)
    应用程序 ID（最小值：0，最大值：无限）
  • AssemblyName (min: 0, max: unbounded)
    程序集名称 (最小值: 0, 最大值: 无限制)
  • Class (min: 0, max: unbounded)
    类 (最小值: 0, 最大值: 无限制)
  • CopyFile (min: 0, max: unbounded): Used to create a duplicate of this file elsewhere.
    复制文件 (最小值: 0, 最大值: 无限制): 用于在其他地方创建此文件的副本。
  • ODBCDriver (min: 0, max: unbounded)
    ODBC 驱动程序（最小值：0，最大值：无限）
  • ODBCTranslator (min: 0, max: unbounded)
    ODBC 翻译器（最小值：0，最大值：无限）
  • Permission (min: 0, max: unbounded): Used to configure the ACLs for this file.
    权限（最小值：0，最大值：无限）：用于配置此文件的 ACL。
  • PermissionEx (min: 0, max: unbounded): Can also configure the ACLs for this file.
    PermissionEx（最小值：0，最大值：无限）：还可以为此文件配置 ACL。
  • Shortcut (min: 0, max: unbounded): Target of the shortcut will be set to this file.
    快捷方式（最小值：0，最大值：无限）：快捷方式的目标将设置为此文件。
  • SymbolPath (min: 0, max: unbounded)
    符号路径（最小值：0，最大值：无限）
  • TypeLib (min: 0, max: unbounded)
    TypeLib（最小值：0，最大值：无限）

    任何元素（namespace='##other' processContents='Lax'）WiX XML Schema 中的扩展点。模式扩展可以在模式中的此点注册其他元素。

Attributes 属性
NameTypeDescriptionRequired
AssemblyEnumeration Specifies if this File is a Win32 Assembly or .NET Assembly that needs to be installed into the Global Assembly Cache (GAC). If the value is '.net' or 'win32', this file must also be the key path of the Component. This attribute's value must be one of the following:
指定此文件是需要安装到全局程序集缓存 (GAC) 的 Win32 程序集或 .NET 程序集。如果值是 '.net' 或 'win32'，则此文件还必须是组件的关键路径。此属性的值必须是以下之一：
.net
The file is a .NET Framework assembly.
该文件是一个 .NET Framework 程序集。
no
The file is not a .NET Framework or Win32 assembly. This is the default value.
该文件不是 .NET Framework 或 Win32 程序集。这是默认值。
win32
The file is a Win32 assembly.
该文件是一个 Win32 程序集。
 
AssemblyApplicationString Specifies the file identifier of the application file. This assembly will be isolated to the same directory as the application file. If this attribute is absent, the assembly will be installed to the Global Assembly Cache (GAC). This attribute may only be specified if the Assembly attribute is set to '.net' or 'win32'.
指定应用程序文件的文件标识符。此程序集将被隔离到与应用程序文件相同的目录中。如果此属性不存在，程序集将安装到全局程序集缓存（GAC）中。仅当 Assembly 属性设置为“.net”或“win32”时，才可以指定此属性。		 
AssemblyManifestString Specifies the file identifier of the manifest file that describes this assembly. The manifest file should be in the same component as the assembly it describes. This attribute may only be specified if the Assembly attribute is set to '.net' or 'win32'.
指定描述此程序集的清单文件的文件标识符。清单文件应与其描述的程序集位于同一组件中。仅当 Assembly 属性设置为“.net”或“win32”时，才可以指定此属性。		 
BindPathStringA list of paths, separated by semicolons, that represent the paths to be searched to find the imported DLLs. The list is usually a list of properties, with each property enclosed inside square brackets. The value may be set to an empty string. Including this attribute will cause an entry to be generated for the file in the BindImage table.
由分号分隔的路径列表，表示要搜索以找到导入的 DLL 的路径。该列表通常是属性列表，每个属性都用方括号括起来。该值可以设置为空字符串。包括此属性将导致在 BindImage 表中为文件生成条目。		 
ChecksumYesNoTypeThis attribute should be set to "yes" for every executable file in the installation that has a valid checksum stored in the Portable Executable (PE) file header. Only those files that have this attribute set will be verified for valid checksum during a reinstall.
对于安装中具有有效校验和存储在可移植可执行（PE）文件头中的每个可执行文件，应将此属性设置为“是”。只有那些设置了此属性的文件在重新安装期间才会验证有效校验和。		 
CompanionFileStringSet this attribute to make this file a companion child of another file. The installation state of a companion file depends not on its own file versioning information, but on the versioning of its companion parent. A file that is the key path for its component can not be a companion file (that means this attribute cannot be set if KeyPath="yes" for this file). The Version attribute cannot be set along with this attribute since companion files are not installed based on their own version.
设置此属性以使此文件成为另一个文件的伴随子文件。伴随文件的安装状态不取决于其自身的文件版本信息，而是取决于其伴随父文件的版本信息。作为其组件的关键路径的文件不能成为伴随文件（这意味着如果 KeyPath="yes"则不能设置此属性）。由于伴随文件不是基于其自身版本安装的，因此不能与 Version 属性一起设置。		 
CompressedYesNoDefaultTypeSets the file's source type compression. A setting of "yes" or "no" will override the setting in the Word Count Summary Property.
设置文件的源类型压缩。设置为“是”或“否”将覆盖字数统计摘要属性中的设置。		 
DefaultLanguageStringThis is the default language of this file. The linker will replace this value from the value in the file if the suppress files option is not used.
这是此文件的默认语言。如果未使用抑制文件选项，链接器将从文件中的值替换此值。		 
DefaultSizeIntegerThis is the default size of this file. The linker will replace this value from the value in the file if the suppress files option is not used.
这是此文件的默认大小。如果未使用抑制文件选项，链接器将从文件中的值替换此值。		 
DefaultVersionStringThis is the default version of this file. The linker will replace this value from the value in the file if the suppress files option is not used.
这是此文件的默认版本。如果不使用抑制文件选项，链接器将用文件中的值替换此值。		 
DiskIdDiskIdType The value of this attribute should correspond to the Id attribute of a Media element authored elsewhere. By creating this connection between a file and its media, you set the packaging options to the values specified in the Media element (values such as compression level, cab embedding, etc...). Specifying the DiskId attribute on the File element overrides the default DiskId attribute from the parent Component element. If no DiskId attribute is specified, the default is "1". This DiskId attribute is ignored when creating a merge module because merge modules do not have media.
此属性的值应对应于其他地方编写的媒体元素的 Id 属性。通过在文件和其媒体之间创建此连接，您可以将打包选项设置为媒体元素中指定的值（如压缩级别、cab 嵌入等）。在文件元素上指定 DiskId 属性将覆盖父组件元素的默认 DiskId 属性。如果未指定 DiskId 属性，则默认值为“1”。在创建合并模块时忽略此 DiskId 属性，因为合并模块没有媒体。		 
FontTitleStringCauses an entry to be generated for the file in the Font table with the specified FontTitle. This attribute is intended to be used to register the file as a non-TrueType font.
在字体表中为文件生成一个具有指定 FontTitle 的条目。此属性旨在用于将文件注册为非 TrueType 字体。		 
HiddenYesNoTypeSet to yes in order to have the file's hidden attribute set when it is installed on the target machine.
设置为是，以便在目标计算机上安装文件时设置文件的隐藏属性。		 
IdString The unique identifier for this File element. If you omit Id, it defaults to the file name portion of the Source attribute, if specified. May be referenced as a Property by specifying [#value].
此文件元素的唯一标识符。如果省略 Id，则默认为 Source 属性的文件名部分（如果指定）。可以通过指定 [#value] 作为属性引用。		 
KeyPathYesNoTypeSet to yes in order to force this file to be the key path for the parent component.
设置为是，以强制此文件成为父组件的关键路径。		 
LongNameLongFileNameTypeThis attribute has been deprecated; please use the Name attribute instead.
此属性已被弃用，请使用 Name 属性。		 
NameLongFileNameType In prior versions of the WiX toolset, this attribute specified the short file name. This attribute's value may now be either a short or long file name. If a short file name is specified, the ShortName attribute may not be specified. If a long file name is specified, the LongName attribute may not be specified. Also, if this value is a long file name, the ShortName attribute may be omitted to allow WiX to attempt to generate a unique short file name. However, if this name collides with another file or you wish to manually specify the short file name, then the ShortName attribute may be specified. Finally, if this attribute is omitted then its default value is the file name portion of the Source attribute, if one is specified, or the value of the Id attribute, if the Source attribute is omitted or doesn't contain a file name.
在以前版本的 WiX 工具集中，此属性指定短文件名。此属性的值现在可以是短文件名或长文件名。如果指定了短文件名，则不能指定 ShortName 属性。如果指定了长文件名，则不能指定 LongName 属性。此外，如果此值是长文件名，可以省略 ShortName 属性，以允许 WiX 尝试生成唯一的短文件名。但是，如果此名称与另一个文件冲突或您希望手动指定短文件名，则可以指定 ShortName 属性。最后，如果省略此属性，则其默认值为 Source 属性的文件名部分（如果指定了 Source 属性），或者 Id 属性的值（如果省略了 Source 属性或不包含文件名）。		 
PatchAllowIgnoreOnErrorYesNoTypeSet to indicate that the patch is non-vital.
设置为指示patch是非重要的。		 
PatchGroupInteger This attribute must be set for patch-added files. Each patch should be assigned a different patch group number. Patch groups numbers must be greater 0 and should be assigned consecutively. For example, the first patch should use PatchGroup='1', the second patch will have PatchGroup='2', etc...
必须为patch添加的文件设置此属性。每个patch应分配不同的patch组号。Patch组号必须大于 0，并应连续分配。例如，第一个patch应使用 PatchGroup='1'，第二个patch将使用 PatchGroup='2'，等等。		 
PatchIgnoreYesNoTypePrevents the updating of the file that is in fact changed in the upgraded image relative to the target images.
防止更新在升级的镜像中相对于目标镜像实际更改的文件。		 
PatchWholeFileYesNoTypeSet if the entire file should be installed rather than creating a binary patch.
如果应安装整个文件而不是创建二进制patch，请设置。		 
ProcessorArchitectureEnumerationSpecifies the architecture for this assembly. This attribute should only be used on .NET Framework 2.0 or higher assemblies. This attribute's value must be one of the following:
指定此程序集的架构。此属性应仅用于 .NET Framework 2.0 或更高版本的程序集。此属性的值必须是以下之一：
msil
The file is a .NET Framework assembly that is processor-neutral.
该文件是一个与处理器无关的 .NET Framework 程序集。
x86
The file is a .NET Framework assembly for the x86 processor.
该文件是一个用于 x86 处理器的 .NET Framework 程序集。
x64
The file is a .NET Framework assembly for the x64 processor.
该文件是用于 x64 处理器的.NET Framework 程序集。
arm
The file is a .NET Framework assembly for a 32-bit ARM processor.
该文件是用于 32 位 ARM 处理器的.NET Framework 程序集。
arm64
The file is a .NET Framework assembly for a 64-bit ARM processor.
该文件是用于 64 位 ARM 处理器的.NET Framework 程序集。
ia64
The file is a .NET Framework assembly for the ia64 processor.
该文件是用于 ia64 处理器的.NET Framework 程序集。
 
ReadOnlyYesNoTypeSet to yes in order to have the file's read-only attribute set when it is installed on the target machine.
设置为是，以便在目标机器上安装文件时设置文件的只读属性。		 
SelfRegCostIntegerThe cost of registering the file in bytes. This must be a non-negative number. Including this attribute will cause an entry to be generated for the file in the SelfReg table.
注册文件的成本（以字节为单位）。这必须是一个非负数。包括此属性将导致在 SelfReg 表中为该文件生成一个条目。		 
ShortNameShortFileNameType The short file name of the file in 8.3 format. This attribute should only be set if there is a conflict between generated short file names or the user wants to manually specify the short file name.
文件的短文件名为 8.3 格式。只有在生成的短文件名之间存在冲突或用户希望手动指定短文件名时，才应设置此属性。		 
SourceStringSpecifies the path to the File in the build process. Overrides default source path set by parent directories and Name attribute. This attribute must be set if no source information can be gathered from parent directories. For more information, see Specifying source files.
指定构建过程中文件的路径。覆盖父目录和名称属性设置的默认源路径。如果无法从父目录中获取源信息，则必须设置此属性。有关更多信息，请参阅指定源文件。		 
srcStringThis attribute has been deprecated; please use the Source attribute instead.
此属性已被弃用；请改用 Source 属性。		 
SystemYesNoTypeSet to yes in order to have the file's system attribute set when it is installed on the target machine.
设置为“是”以便在目标机器上安装文件时设置文件的系统属性。		 
TrueTypeYesNoTypeCauses an entry to be generated for the file in the Font table with no FontTitle specified. This attribute is intended to be used to register the file as a TrueType font.
使文件在字体表中生成一个没有指定 FontTitle 的条目。此属性旨在将文件注册为 TrueType 字体。		 
VitalYesNoTypeIf a file is vital, then installation cannot proceed unless the file is successfully installed. The user will have no option to ignore an error installing this file. If an error occurs, they can merely retry to install the file or abort the installation. The default is "yes," unless the -sfdvital switch (candle.exe) or SuppressFileDefaultVital property (.wixproj) is used.
如果文件是至关重要的，则除非文件成功安装，否则安装无法继续。用户将无法选择忽略安装此文件时的错误。如果发生错误，他们只能重试安装文件或中止安装。默认值为“是”，除非使用-sfdvital 开关（candle.exe）或 SuppressFileDefaultVital 属性（.wixproj）。		 
Any Attribute (namespace='##other' processContents='lax') Extensibility point in the WiX XML Schema. Schema extensions can register additional attributes at this point in the schema.
任何属性（namespace='##other' processContents='lax'）在 WiX XML 模式中的可扩展性点。模式扩展可以在模式中的此点注册其他属性。
