Directory Element 目录元素

Description 描述
Directory layout for the product. Also specifies the mappings between source and target directories.
产品的目录布局。还指定了源目录和目标目录之间的映射。
Windows Installer references

Attributes 属性
NameTypeDescriptionRequired
IdStringThis value is the unique identifier of the directory entry.
此值是目录条目的唯一标识符。		Yes
ComponentGuidGenerationSeedGuid The Component Guid Generation Seed is a guid that must be used when a Component with the generate guid directive ("*") is not rooted in a standard Windows Installer directory (for example, ProgramFilesFolder or CommonFilesFolder). It is recommended that this attribute be avoided and that developers install their Components under standard directories with unique names instead (for example, "ProgramFilesFolder\Company Name Product Name Version"). It is important to note that once a directory is assigned a Component Guid Generation Seed the value must not change until (and must be changed when) the path to that directory, including itself and all parent directories, changes.
组件 GUID 生成种子是一个 GUID，当组件使用生成 GUID 指令（“*”）且不在标准 Windows Installer 目录（例如 ProgramFilesFolder 或 CommonFilesFolder）中时，必须使用该 GUID。建议避免使用此属性，开发人员应将其组件安装在具有唯一名称的标准目录下（例如，“ProgramFilesFolder\公司名称 产品名称 版本”）。重要的是，一旦目录分配了组件 GUID 生成种子，该值在目录路径（包括自身及所有父目录）更改之前不得更改，并且在路径更改时必须更改。		 
DiskIdDiskIdType Sets the default disk identifier for the files contained in this directory. This attribute's value may be overridden by a child Component, Directory, Merge or File element. See the File or Merge elements' DiskId attribute for more information.
设置此目录中包含的文件的默认磁盘标识符。此属性的值可能会被子组件、目录、合并或文件元素覆盖。有关文件或合并元素的 DiskId 属性的更多信息，请参见文件或合并元素的 DiskId 属性。		 
FileSourceStringUsed to set the file system source for this directory's child elements. For more information, see Specifying source files.
用于设置此目录的子元素的文件系统源。有关更多信息，请参见指定源文件。		 
LongNameLongFileNameTypeThis attribute has been deprecated; please use the Name attribute instead.
此属性已被弃用，请使用 Name 属性。		 
LongSourceLongFileNameTypeThis attribute has been deprecated; please use the SourceName attribute instead.
此属性已被弃用，请使用 SourceName 属性。		 
NameLongFileNameType The name of the directory.
目录的名称。

Do not specify this attribute (or the LongName attribute) if this directory represents the same directory as the parent (see the Windows Installer SDK's Directory table topic for more information about the "." operator).
如果此目录与父目录相同，请不要指定此属性（或 LongName 属性）（有关“.”运算符的更多信息，请参阅 Windows Installer SDK 的目录表主题）。

In prior versions of the WiX toolset, this attribute specified the short directory name. This attribute's value may now be either a short or long directory name. If a short directory name is specified, the ShortName attribute may not be specified. If a long directory name is specified, the LongName attribute may not be specified. Also, if this value is a long directory name, the ShortName attribute may be omitted to allow WiX to attempt to generate a unique short directory name. However, if this name collides with another directory or you wish to manually specify the short directory name, then the ShortName attribute may be specified.
在以前版本的 WiX 工具集中，此属性指定短目录名称。此属性的值现在可以是短目录名称或长目录名称。如果指定了短目录名称，则不能指定 ShortName 属性。如果指定了长目录名称，则不能指定 LongName 属性。此外，如果此值是长目录名称，则可以省略 ShortName 属性，以允许 WiX 尝试生成唯一的短目录名称。但是，如果此名称与另一个目录冲突或您希望手动指定短目录名称，则可以指定 ShortName 属性。		 
ShortNameShortFileNameType The short name of the directory in 8.3 format. This attribute should only be set if there is a conflict between generated short directory names or the user wants to manually specify the short directory name.
目录的短名称为 8.3 格式。只有在生成的短目录名称之间存在冲突或用户希望手动指定短目录名称时，才应设置此属性。		 
ShortSourceNameShortFileNameType The short name of the directory on the source media in 8.3 format. This attribute should only be set if there is a conflict between generated short directory names or the user wants to manually specify the short source directory name.
源媒体上目录的短名称为 8.3 格式。只有在生成的短目录名称之间存在冲突或用户希望手动指定短源目录名称时，才应设置此属性。		 
SourceNameLongFileNameType The name of the directory on the source media. If this attribute is not specified, Windows Installer will default to the Name attribute.
源媒体上目录的名称。如果未指定此属性，Windows Installer 将默认为 Name 属性。

In prior versions of the WiX toolset, this attribute specified the short source directory name. This attribute's value may now be either a short or long directory name. If a short directory name is specified, the ShortSourceName attribute may not be specified. If a long directory name is specified, the LongSource attribute may not be specified. Also, if this value is a long directory name, the ShortSourceName attribute may be omitted to allow WiX to attempt to generate a unique short directory name. However, if this name collides with another directory or you wish to manually specify the short directory name, then the ShortSourceName attribute may be specified.
在以前版本的 WiX 工具集中，此属性指定了短源目录名称。此属性的值现在可以是短目录名称或长目录名称。如果指定了短目录名称，则不能指定 ShortSourceName 属性。如果指定了长目录名称，则不能指定 LongSource 属性。此外，如果此值是长目录名称，则可以省略 ShortSourceName 属性，以允许 WiX 尝试生成唯一的短目录名称。但是，如果此名称与另一个目录冲突或您希望手动指定短目录名称，则可以指定 ShortSourceName 属性。		 
srcStringThis attribute has been deprecated; please use the FileSource attribute instead.
此属性已被弃用，请使用 FileSource 属性代替。
