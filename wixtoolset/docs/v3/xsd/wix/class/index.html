Class Element 类元素

Description 描述
COM Class registration for parent Component.
父组件的 COM 类注册。
Windows Installer references
Windows 安装程序引用
Class Table, ProgId Table, Registry Table, AppId Table
类表、ProgId 表、注册表、AppId 表
Parents 父母
AppId, Component, File, TypeLib
应用程序 ID, 组件, 文件, 类型库
Inner Text 内部文本
None 无
Children 儿童
Choice of elements (min: 0, max: unbounded)
元素选择（最小值：0，最大值：无限）
  • FileTypeMask (min: 0, max: unbounded)
    文件类型掩码 (最小: 0, 最大: 无限制)
  • Interface (min: 0, max: unbounded): These Interfaces will be registered with the parent Class and TypeLib (if present).
    接口（最小值：0，最大值：无限）：这些接口将注册到父类和类型库（如果存在）。
  • ProgId (min: 0, max: unbounded): A ProgId associated with Class must be a child element of the Class element
    ProgId（最小值：0，最大值：无限）：与类关联的 ProgId 必须是类元素的子元素。
Attributes 属性
NameTypeDescriptionRequired
IdGuidThe Class identifier (CLSID) of a COM server.
COM 服务器的类标识符（CLSID）。		Yes
AdvertiseYesNoType Set this value to "yes" in order to create a normal Class table row. Set this value to "no" in order to generate Registry rows that perform similar registration (without the often problematic Windows Installer advertising behavior).
将此值设置为“是”以创建正常的类表行。将此值设置为“否”以生成执行类似注册的注册表行（没有通常有问题的 Windows Installer 广告行为）。		 
AppIdGuid This attribute is only allowed when a Class is advertised. Using this attribute will reference an Application ID containing DCOM information for the associated application GUID. The value must correspond to an AppId/@Id of an AppId element nested under a Fragment, Module, or Product element. To associate an AppId with a non-advertised class, nest the class within a parent AppId element.
仅当类被广告时才允许使用此属性。使用此属性将引用包含关联应用程序 GUID 的 DCOM 信息的应用程序 ID。该值必须对应于嵌套在 Fragment、Module 或 Product 元素下的 AppId 元素的 AppId/@Id。要将 AppId 与非广告类关联，请将类嵌套在父 AppId 元素中。		 
ArgumentString This column is optional only when the Context column is set to "LocalServer" or "LocalServer32" server context. The text is registered as the argument against the OLE server and is used by OLE for invoking the server. Note that the resolution of properties in the Argument field is limited. A property formatted as [Property] in this field can only be resolved if the property already has the intended value when the component owning the class is installed. For example, for the argument "[#MyDoc.doc]" to resolve to the correct value, the same process must be installing the file MyDoc.doc and the component that owns the class.
仅当 Context 列设置为“LocalServer”或“LocalServer32”服务器上下文时，此列才是可选的。文本作为参数注册到 OLE 服务器，并由 OLE 用于调用服务器。请注意，Argument 字段中的属性解析是有限的。格式为[Property]的属性只有在类所属的组件安装时该属性已经具有预期值时才能解析。例如，要使参数“[#MyDoc.doc]”解析为正确的值，必须由同一进程安装文件 MyDoc.doc 和拥有该类的组件。		 
ContextList The server context(s) for this COM server. This attribute is optional for VB6 libraries that are marked "PublicNotCreateable". Class elements marked Advertised must specify at least one server context. It is most common for there to be a single value for the Context attribute. This attribute's value should be a space-delimited list containg one or more of the following:
此 COM 服务器的服务器上下文。对于标记为“PublicNotCreateable”的 VB6 库，此属性是可选的。标记为 Advertised 的类元素必须指定至少一个服务器上下文。通常情况下，Context 属性只有一个值。此属性的值应为包含以下一个或多个内容的以空格分隔的列表：
LocalServer 本地服务器
A 16-bit local server application.
一个 16 位的本地服务器应用程序。
LocalServer32 本地服务器 32
A 32-bit local server application.
一个 32 位本地服务器应用程序。
InprocServer 内部服务器
A 16-bit in-process server DLL.
一个 16 位的进程内服务器 DLL。
InprocServer32
A 32-bit in-process server DLL.
一个 32 位的进程内服务器 DLL。
 
ControlYesNoType Set this attribute's value to 'yes' to identify an object as an ActiveX Control. The default value is 'no'.
将此属性的值设置为“yes”以将对象标识为 ActiveX 控件。默认值为“no”。		 
DescriptionStringLocalized description associated with the Class ID and Program ID.
与类 ID 和程序 ID 相关的本地化描述。		 
ForeignServerString May only be specified if the value of the Advertise attribute is "no" and Server has not been specified. In addition, it may only be used when the Class element is directly under the Component element. The value can be that of an registry type (REG_SZ). This attribute should be used to specify foreign servers, such as mscoree.dll if needed.
只有在 Advertise 属性的值为“no”且未指定 Server 时才可指定。此外，它只能在 Class 元素直接位于 Component 元素下时使用。该值可以是注册表类型（REG_SZ）的值。此属性应用于指定外部服务器，例如 mscoree.dll（如果需要）。		 
HandlerString The default inproc handler. May be optionally provided only for Context = LocalServer or LocalServer32. Value of "1" creates a 16-bit InprocHandler (appearing as the InprocHandler value). Value of "2" creates a 32-bit InprocHandler (appearing as the InprocHandler32 value). Value of "3" creates 16-bit as well as 32-bit InprocHandlers. A non-numeric value is treated as a system file that serves as the 32-bit InprocHandler (appearing as the InprocHandler32 value).
默认的进程内处理程序。仅可选地为 Context = LocalServer 或 LocalServer32 提供。"1"的值创建一个 16 位的 InprocHandler（显示为 InprocHandler 值）。"2"的值创建一个 32 位的 InprocHandler（显示为 InprocHandler32 值）。"3"的值创建 16 位和 32 位的 InprocHandler。非数字值被视为作为 32 位 InprocHandler 的系统文件（显示为 InprocHandler32 值）。		 
IconString The file providing the icon associated with this CLSID. Reference to an Icon element (should match the Id attribute of an Icon element). This is currently not supported if the value of the Advertise attribute is "no".
提供与此 CLSID 关联的图标的文件。引用一个 Icon 元素（应匹配 Icon 元素的 Id 属性）。如果 Advertise 属性的值为"no"，则当前不支持此功能。		 
IconIndexIntegerIcon index into the icon file.
图标文件中的图标索引。		 
InsertableYesNoType Specifies the CLSID may be insertable.
指定 CLSID 可能是可插入的。		 
ProgrammableYesNoType Specifies the CLSID may be programmable.
指定 CLSID 可能是可编程的。		 
RelativePathYesNoType When the value is "yes", the bare file name can be used for COM servers. The installer registers the file name only instead of the complete path. This enables the server in the current directory to take precedence and allows multiple copies of the same component.
当值为“是”时，可以使用裸文件名来表示 COM 服务器。安装程序仅注册文件名而不是完整路径。这使得当前目录中的服务器优先，并允许同一组件的多个副本。		 
SafeForInitializingYesNoType May only be specified if the value of the Advertise attribute is "no".
只有在 Advertise 属性的值为“no”时才能指定。		 
SafeForScriptingYesNoType May only be specified if the value of the Advertise attribute is "no".
只有在 Advertise 属性的值为“no”时才能指定。		 
ServerString May only be specified if the value of the Advertise attribute is "no" and the ForeignServer attribute is not specified. File Id of the COM server file. If this element is nested under a File element, this value defaults to the value of the parent File/@Id.
只有在 Advertise 属性的值为“no”且未指定 ForeignServer 属性时才能指定。COM 服务器文件的文件 ID。如果此元素嵌套在 File 元素下，则此值默认为父 File/@Id 的值。		 
ShortPathYesNoType Specifies whether or not to use the short path for the COM server. This can only apply when Advertise is set to 'no'. The default is 'no' meaning that it will use the long file name for the COM server.
指定是否使用 COM 服务器的短路径。只有在 Advertise 设置为“否”时才适用。默认值为“否”，这意味着它将使用 COM 服务器的长文件名。		 
ThreadingModelEnumeration Threading model for the CLSID. This attribute's value must be one of the following:
CLSID 的线程模型。此属性的值必须是以下之一：
apartment 单元
free 自由
both 两者
neutral 中立
single 单一
rental 租赁
 
VersionString Version for the CLSID.
CLSID 的版本。		 
Remarks 备注
When being used in unadvertised mode, the attributes in the Class element correspond to registry keys as follows (values that can be specified in authoring are in bold):
在未公开模式下使用时，Class 元素中的属性对应于注册表项如下（可以在创作中指定的值以粗体显示）：
Id/Context/Server
In General 一般来说
[HKCR\CLSID\{Id}\Context1]
@="[!Server]"
[HKCR\CLSID\{Id}\Context2]
@="[!Server]"
Specific Example 具体示例
[HKCR\CLSID\{01234567-89AB-CDEF-0123-456789ABCDEF}\LocalServer]
@="[!comserv.dll]"
[HKCR\CLSID\{01234567-89AB-CDEF-0123-456789ABCDEF}\LocalServer32]
@="[!comserv.dll]"
Id/Context/ForeignServer
In General 一般来说
[HKCR\CLSID\{Id}\Context1]
@="ForeignServer"
[HKCR\CLSID\{Id}\Context2]
@="ForeignServer"
Specific Example 具体示例

@="mscoree.dll"
[HKCR\CLSID\{01234567-89AB-CDEF-0123-456789ABCDEF}\LocalServer32]
@="mscoree.dll"
