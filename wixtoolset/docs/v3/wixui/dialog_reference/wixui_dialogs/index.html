WixUI Dialogs WixUI 对话框

The following table describes each of the built-in dialogs that is defined in the WixUI dialog library.
下表描述了在 WixUI 对话框 library 中定义的每个内置对话框。

Dialog Name 对话框名称Description
AdvancedWelcomeEulaDlgA dialog that displays the end user license agreement. Unlike the LicenseAgreementDlg, it has Advanced and Install buttons instead of Next and Back buttons. This dialog is used by the WixUI_Advanced dialog set to provide the user with a quick way to perform a default installation.
显示最终用户许可协议的对话框。与 LicenseAgreementDlg 不同，它有高级和安装按钮，而不是下一步和返回按钮。此对话框由 WixUI_Advanced 对话框集使用，为用户提供快速执行默认安装的方法。
BrowseDlgA dialog that allows the user to browse for a destination folder.
允许用户浏览目标文件夹的对话框。
CancelDlgA dialog that appears after the user clicks a Cancel button on any dialog and confirms whether or not the user really wants to cancel the installation.
用户在任何对话框上点击取消按钮后出现的对话框，确认用户是否真的想取消安装。
CustomizeDlgA dialog that displays a feature selection tree with a Browse button, Disk Usage button, and a text box that contains information about the currently selected feature.
显示功能选择树的对话框，带有浏览按钮、磁盘使用按钮和包含当前选定功能信息的文本框。
DiskCostDlgA dialog that allows the user to select which drive to install to and that displays disk space usage information for each drive.
允许用户选择安装驱动器并显示每个驱动器磁盘空间使用信息的对话框。
ErrorDlgA dialog that displays an error message to the user and can provide an option to retry the previous action.
显示错误信息并提供重试上一个操作选项的对话框。
ExitDlgA dialog that displays a summary dialog after setup completes successfully. It can also optionally display a checkbox and custom text. For details about how to add a checkbox and custom text to this dialog, see [Customizing Built-in WixUI Dialog Sets](/wixtoolset/docs/v3/wixui/wixui_customizations) and [How To: Run the Installed Application After Setup](/wixtoolset/docs/v3/howtos/ui_and_localization/run_program_after_install).
一个对话框，在设置成功完成后显示摘要对话框。它还可以选择显示一个复选框和自定义文本。有关如何向此对话框添加复选框和自定义文本的详细信息，请参阅[自定义内置 WixUI 对话框集](/wixtoolset/docs/v3/wixui/wixui_customizations)和[如何：在设置后运行已安装的应用程序](/wixtoolset/docs/v3/howtos/ui_and_localization/run_program_after_install)。
FatalErrorA dialog that displays a summary error dialog if setup fails.
一个对话框，如果设置失败，则显示摘要错误对话框。
FeaturesDlgA dialog that displays a feature selection tree with a text box that contains information about the currently selected feature. Unlike the CustomizeDlg, it does not contain Browse or Disk Space buttons.
一个对话框，显示一个功能选择树，并带有一个文本框，其中包含有关当前选择的功能的信息。与 CustomizeDlg 不同，它不包含浏览或磁盘空间按钮。
FilesInUseA dialog that displays a list of applications that are holding files in use that need to be updated by the current installation process. It includes Retry, Ignore and Exit buttons.
显示一个对话框，其中列出了当前安装过程中需要更新的正在使用文件的应用程序列表。它包括重试、忽略和退出按钮。
InstallDirDlgA dialog that has a text box that allows the user to type in a non-default installation path and a Browse button that allows the user to select a non-default installation folder. By default, the InstallDirDlg dialog validates that any path the user enters is valid for Windows Installer: That is, it's a path on a local hard drive, not a network path or on a removable drive. If you wish to disable path validation and allow invalid paths, set the public property WIXUI_DONTVALIDATEPATH to 1.
一个对话框，包含一个文本框，允许用户输入非默认的安装路径，并有一个浏览按钮，允许用户选择非默认的安装文件夹。默认情况下，InstallDirDlg 对话框会验证用户输入的任何路径是否对 Windows Installer 有效：即，它是本地硬盘上的路径，而不是网络路径或可移动驱动器上的路径。如果您希望禁用路径验证并允许无效路径，请将公共属性 WIXUI_DONTVALIDATEPATH 设置为 1。
InstallScopeDlgA dialog that allows the user to choose to install the product for all users or for the current user.
一个对话框，允许用户选择为所有用户或当前用户安装产品。
InvalidDirDlgA dialog that displays an error if the user selects an invalid installation directory.
如果用户选择了无效的安装目录，将显示一个错误对话框。
LicenseAgreementDlgA dialog that displays the end user license agreement and includes Back and Next buttons. Unlike the AdvancedWelcomeEulaDlg, this dialog does not allow the user to start a default installation.
显示最终用户许可协议并包含“返回”和“下一步”按钮的对话框。与 AdvancedWelcomeEulaDlg 不同，此对话框不允许用户开始默认安装。
MaintenanceTypeDlgA dialog that includes buttons that allow the user to change which features are installed, repair the product or remove the product. It only appears when the user runs setup after a product has been installed.
包含按钮的对话框，允许用户更改安装的功能、修复产品或删除产品。仅在用户在产品安装后运行设置时出现。
MaintenanceWelcomeDlgAn introductory dialog that appears when running setup after the product has been installed.
安装产品后运行设置时出现的介绍性对话框。
MsiRMFilesInUseA dialog that is similar to the FilesInUse dialog, but that interacts with Restart Manager. It allows the user to attempt to automatically close applications or ignore the prompt and result in the setup requiring a reboot after it completes.
一个类似于 FilesInUse 对话框的对话框，但与重启管理器交互。它允许用户尝试自动关闭应用程序或忽略提示，导致设置完成后需要重新启动。
OutOfDiskDlgA dialog that informs the user that they have insufficient disk space on the selected drive and advises them to free up additional disk space or reduce the number of features to be installed to the drive.
一个通知用户所选驱动器上磁盘空间不足的对话框，并建议他们释放更多磁盘空间或减少要安装到驱动器上的功能数量。
OutOfRbDiskDlgA dialog that is similar to the OutOfDiskDlg, but also allows the user to disable Windows Installer rollback functionality in order to conserve disk space required by setup.
一个类似于 OutOfDiskDlg 的对话框，但也允许用户禁用 Windows Installer 回滚功能，以节省安装所需的磁盘空间。
PrepareDlgA simple progress dialog that appears during setup initialization before the first interactive dialog appears.
一个简单的进度对话框，在第一个交互对话框出现之前的安装初始化期间出现。
ProgressDlgA dialog that appears during installation that displays a progress bar and messages about actions are being performed.
安装过程中出现的对话框，显示进度条和有关正在执行的操作的消息。
ResumeDlgAn introductory dialog that appears when resuming a suspended setup.
恢复暂停的设置时出现的介绍对话框。
SetupTypeDlgA dialog that allows the user to choose Typical, Custom or Complete installation configurations.
允许用户选择典型、自定义或完整安装配置的对话框。
UserExitA dialog that that is similar to the FatalError dialog. It displays a summary dialog if the user chooses to cancel setup.
类似于 FatalError 对话框的对话框。如果用户选择取消设置，它会显示一个摘要对话框。
VerifyReadyDlgA dialog that appears immediately before starting installation. It asks the user for final confirmation before starting to make changes to the system.
在开始安装之前立即出现的对话框。它要求用户在开始对系统进行更改之前进行最终确认。
WaitForCostingDlgA dialog that appears if the user advances too far in the setup wizard before Windows Installer has finished calculating disk cost requirements.
如果用户在 Windows Installer 完成磁盘成本计算之前在安装向导中前进得太远，则会出现一个对话框。
WelcomeDlgAn introductory dialog that appears when running setup for a product that has not yet been installed.
当运行尚未安装的产品的安装程序时，会出现一个介绍性对话框。
WelcomeEulaDlgA dialog that displays an end user license agreement and allows the user to start installation after accepting the agreement. It is only used by the WixUI_Minimal dialog set and is intended for simple setup programs that do not offer any user configurable options.
显示最终用户许可协议的对话框，并允许用户在接受协议后开始安装。它仅由 WixUI_Minimal 对话框集使用，旨在用于不提供任何用户可配置选项的简单安装程序。
