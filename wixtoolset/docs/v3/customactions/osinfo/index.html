OSInfo custom actions OS 信息自定义操作

The WixQueryOsInfo, WixQueryOsDirs, WixQueryOsDriverInfo, and WixQueryNativeMachine custom actions in wixca (part of WixUtilExtension) set properties over and above the MSI set for OS product/suite detection, architecture, and standard directories. The WixQueryOsWellKnownSID custom action sets properties for the localized names of some built in Windows users and groups.
WixQueryOsInfo、WixQueryOsDirs、WixQueryOsDriverInfo 和 WixQueryNativeMachine 自定义操作在 wixca（WixUtilExtension 的一部分）中设置了超出 MSI 设置的属性，用于操作系统产品/套件检测、架构和标准目录。WixQueryOsWellKnownSID 自定义操作设置了一些内置 Windows 用户和组的本地化名称的属性。

To use these custom actions you simply need to add a PropertyRef to the property you want to use and then include WixUtilExtensions when linking. For example:
要使用这些自定义操作，您只需添加一个 PropertyRef 到您想要使用的属性，然后在链接时包含 WixUtilExtensions。例如：

<PropertyRef Id="WIX_SUITE_SINGLEUSERTS" />
<PropertyRef Id="WIX_DIR_COMMON_DOCUMENTS" />
<PropertyRef Id="WIX_ACCOUNT_LOCALSERVICE" />

WixUtilExtension will automatically schedule the custom actions as needed after the AppSearch standard action. For additional information about standard directory tokens in Windows and which ones are supported directly by Windows Installer, see the following topics in the MSDN documentation:
WixUtilExtension 将在 AppSearch 标准操作之后自动安排所需的自定义操作。有关 Windows 中的标准目录标记及其直接由 Windows Installer 支持的详细信息，请参阅 MSDN 文档中的以下主题：

  • Constant special item ID list (CSIDL) values
    常量特殊项目 ID 列表（CSIDL）值
  • Windows Installer system folder values
    Windows Installer 系统文件夹值

WixQueryOsDriverInfo properties
WixQueryOsDriverInfo 属性

WIX_WDDM_DRIVER_PRESENT

Set to 1 if the video card driver on the target machine is a WDDM driver. This property is only set on machines running Windows Vista or higher.
如果目标机器上的视频卡驱动程序是 WDDM 驱动程序，则设置为 1。此属性仅在运行 Windows Vista 或更高版本的机器上设置。

WIX_DWM_COMPOSITION_ENABLEDSet to 1 if the target machine has composition enabled. This property is only set on machines running Windows Vista or higher.
如果目标机器启用了组合，则设置为 1。此属性仅在运行 Windows Vista 或更高版本的机器上设置。

WixQueryNativeMachine properties
WixQueryNativeMachine 属性

WIX_NATIVE_MACHINE

Set to an [Image File Machine value](https://docs.microsoft.com/en-us/windows/win32/sysinfo/image-file-machine-constants) representing the native architecture of the machine. This property was added in v3.14 and is only set on Windows 10, version 1511 (TH2) and higher.
设置为表示机器本机架构的[图像文件机器值](https://docs.microsoft.com/en-us/windows/win32/sysinfo/image-file-machine-constants)。此属性在 v3.14 中添加，仅在 Windows 10 版本 1511（TH2）及更高版本中设置。
