Author Property​ 作者属性

Gets or sets the Author summary information property.
获取或设置作者摘要信息属性。

Declaration​ 声明

public string Author { get; set; }

Remarks​ 备注

The Author summary information property conveys to a file browser the manufacturer of the installation database. This value is typically set from the installer property Manufacturer. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
作者摘要信息属性向文件浏览器传达安装数据库的制造商。此值通常从安装程序属性制造商设置。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

CharacterCount Property​
字符数属性

Gets or sets the CharacterCount summary information property.
获取或设置 CharacterCount 摘要信息属性。

Declaration​ 声明

public int CharacterCount { get; set; }

Remarks​ 备注

The CharacterCount summary information property is only used in transforms. This part of the summary information stream is divided into two 16-bit words. The upper word contains the transform validation flags. The lower word contains the transform error condition flags. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
CharacterCount 摘要信息属性仅在转换中使用。摘要信息流的这一部分分为两个 16 位字。上半部分包含转换验证标志。下半部分包含转换错误条件标志。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

CodePage Property​ 代码页属性

Gets or sets the CodePage summary information property.
获取或设置 CodePage 摘要信息属性。

Declaration​ 声明

public System.Int16 CodePage { get; set; }

Remarks​ 备注

The Codepage summary information property is the numeric value of the ANSI code page used for any strings that are stored in the summary information. Note that this is not the same code page for strings in the installation database. The Codepage summary information property is used to translate the strings in the summary information into Unicode when calling the Unicode API functions. The Codepage summary information property must be set before any string properties are set in the summary information. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
代码页摘要信息属性是用于存储在摘要信息中的任何字符串的 ANSI 代码页的数值。请注意，这与安装数据库中的字符串代码页不同。代码页摘要信息属性用于在调用 Unicode API 函数时将摘要信息中的字符串转换为 Unicode。代码页摘要信息属性必须在设置摘要信息中的任何字符串属性之前设置。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

Comments Property​ 注释属性

Gets or sets the Comments summary information property.
获取或设置注释摘要信息属性。

Declaration​ 声明

public string Comments { get; set; }

Remarks​ 备注

The Comments summary information property conveys the general purpose of the installer database. By convention, the value for this summary property is set to the following: "This installer database contains the logic and data required to install {'<'}product name{'>'}." where {'<'}product name{'>'} is the name of the product being installed. In general the value for this summary property only changes in the product name, nothing else. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
注释摘要信息属性传达了安装程序数据库的一般用途。按照惯例，此摘要属性的值设置为以下内容：“此安装程序数据库包含安装{'<'}产品名称{'>'}所需的逻辑和数据。”其中{'<'}产品名称{'>'}是正在安装的产品的名称。通常，此摘要属性的值仅在产品名称上发生变化，其他内容不变。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

获取或设置关键词摘要信息属性。

Declaration​ 声明

public string Keywords { get; set; }

Remarks​ 备注

The Keywords summary information property is used by file browsers to hold keywords that permit the database file to be found in a keyword search. The set of keywords typically includes "Installer" as well as product-specific keywords, and may be localized. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
关键字摘要信息属性由文件浏览器用于保存关键字，这些关键字允许在关键字搜索中找到数据库文件。关键字集通常包括“安装程序”以及特定产品的关键字，并且可能是本地化的。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

Declaration​ 声明

public System.DateTime LastPrintTime { get; set; }

Remarks​ 备注

The LastPrintTime summary information property can be set to the date and time during an administrative installation to record when the administrative image was created. For non-administrative installations this property is the same as the CreateTime summary information property. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
LastPrintTime 摘要信息属性可以在管理安装期间设置为日期和时间，以记录创建管理映像的时间。对于非管理安装，此属性与 CreateTime 摘要信息属性相同。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

LastSavedBy Property​ LastSavedBy 属性

Gets or sets the LastSavedBy summary information property.
获取或设置 LastSavedBy 摘要信息属性。

Declaration​ 声明

public string LastSavedBy { get; set; }

Remarks​ 备注

The installer sets the Last Saved By summary information property to the value of the LogonUser property during an administrative installation. The installer never uses this property and a user never needs to modify it. Developers of a database editing tool may use this property to track the last person to modify the database. This property should be left set to null in a final shipping database. In a transform, this summary property contains the platform and language ID(s) that a database should have after it has been transformed. The property specifies to what the Template should be set in the new database. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
安装程序在管理安装期间将“最后保存者”摘要信息属性设置为 LogonUser 属性的值。安装程序从不使用此属性，用户也不需要修改它。数据库编辑工具的开发人员可以使用此属性来跟踪最后修改数据库的人。此属性在最终发布的数据库中应保持为空。在转换中，此摘要属性包含数据库在转换后应具有的平台和语言 ID。该属性指定新数据库中的模板应设置为何值。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

LastSaveTime Property​ 最后保存时间属性

Gets or sets the LastSaveTime summary information property.
获取或设置最后保存时间摘要信息属性。

Declaration​ 声明

public System.DateTime LastSaveTime { get; set; }

Remarks​ 备注

The LastSaveTime summary information property conveys when the last time the installer database was modified. Each time a user changes an installation the value for this summary property is updated to the current system time/date at the time the installer database was saved. Initially the value for this summary property is set to null to indicate that no changes have yet been made. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
LastSaveTime 摘要信息属性传达了安装程序数据库上次修改的时间。每次用户更改安装时，此摘要属性的值都会更新为安装程序数据库保存时的当前系统时间/日期。最初，此摘要属性的值设置为 null，以表示尚未进行任何更改。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

PageCount Property​ 页数属性

Gets or sets the PageCount summary information property.
获取或设置 PageCount 摘要信息属性。

Declaration​ 声明

public int PageCount { get; set; }

Remarks​ 备注

For an installation package, the PageCount summary information property contains the minimum installer version required. For Windows Installer version 1.0, this property must be set to the integer 100. For 64-bit Windows Installer Packages, this property must be set to the integer 200. For a transform package, the PageCount summary information property contains minimum installer version required to process the transform. Set to the greater of the two PageCount summary information property values belonging to the databases used to generate the transform. The PageCount summary information property is set to null in patch packages. This summary property is REQUIRED. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
对于安装包，PageCount 摘要信息属性包含所需的最低安装程序版本。对于 Windows Installer 版本 1.0，此属性必须设置为整数 100。对于 64 位 Windows Installer 包，此属性必须设置为整数 200。对于转换包，PageCount 摘要信息属性包含处理转换所需的最低安装程序版本。设置为生成转换所用数据库的两个 PageCount 摘要信息属性值中较大的一个。在patch包中，PageCount 摘要信息属性设置为 null。此摘要属性是必需的。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

RevisionNumber Property​
修订号属性

Gets or sets the RevisionNumber summary information property.
获取或设置 RevisionNumber 摘要信息属性。

Declaration​ 声明

public string RevisionNumber { get; set; }

Remarks​ 备注

The Revision Number summary information property contains the package code for the installer package. The package code is a unique identifier of the installer package. The Revision Number summary information property of a patch package specifies the GUID patch code for the patch. This is followed by a list of patch code GUIDs for obsolete patches that are removed when this patch is applied. The patch codes are concatenated with no delimiters separating GUIDs in the list. The Revision Number summary information property of a transform package lists the product code GUIDs and version of the new and original products and the upgrade code GUID. The list is separated with semicolons as follows. Original-Product-Code Original-Product-Version ; New-Product Code New-Product-Version; Upgrade-Code This summary property is REQUIRED. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
修订号摘要信息属性包含安装程序包的包代码。包代码是安装程序包的唯一标识符。patch包的修订号摘要信息属性指定patch代码的 GUID patch。接下来是一个patch代码 GUID 列表，用于删除应用此patch时移除的过时补丁。patch代码被连接在一起，没有分隔符分隔列表中的 GUID。转换包的修订号摘要信息属性列出了新旧产品的产品代码 GUID 和版本以及升级代码 GUID。列表用分号分隔，如下所示。原始产品代码 原始产品版本 ; 新产品代码 新产品版本; 升级代码 此摘要属性是必需的。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

Security Property​ 安全属性

Gets or sets the Security summary information property.
获取或设置安全摘要信息属性。

Declaration​ 声明

public int Security { get; set; }

Remarks​ 备注

The Security summary information property conveys whether the package should be opened as read-only. The database editing tool should not modify a read-only enforced database and should issue a warning at attempts to modify a read-only recommended database. The following values of this property are applicable to Windows Installer files:
安全摘要信息属性传达了包是否应以只读方式打开。数据库编辑工具不应修改强制只读的数据库，并且在尝试修改建议只读的数据库时应发出警告。此属性的以下值适用于 Windows Installer 文件：

  • 0 - no restriction 0 - 无限制
  • 2 - read only recommended
    2 - 建议只读
  • 4 - read only enforced
    4 - 只读强制

This property should be set to read-only recommended (2) for an installation database and to read-only enforced (4) for a transform or patch. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
此属性应设置为安装数据库的只读推荐（2），并为转换或patch设置为只读强制（4）。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

Subject Property​ 主题属性

Gets or sets the Subject summary information property.
获取或设置主题摘要信息属性。

Declaration​ 声明

public string Subject { get; set; }

Remarks​ 备注

The Subject summary information property conveys to a file browser the product that can be installed using the logic and data in this installer database. For example, the value of the summary property for Microsoft Office 97 would be "Microsoft Office 97 Professional". This value is typically set from the installer property ProductName. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
主题摘要信息属性向文件浏览器传达可以使用此安装程序数据库中的逻辑和数据安装的产品。例如，Microsoft Office 97 的摘要属性值将是“Microsoft Office 97 Professional”。此值通常从安装程序属性 ProductName 设置。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

If this is a 64-bit Windows Installer, enter Intel64 in the Template summary information property. Note that an installation package cannot have both the Intel and Intel64 properties set. If the current platform does not match one of the platforms specified then the installer will not process the package. Not specifying a platform implies that the package is platform-independent. Entering 0 in the language ID field of the Template summary information property, or leaving this field empty, indicates that the package is language neutral. There are variations of this property depending on whether it is in a source installer database or a transform. Source Installer Database - Only one language can be specified in a source installer database. Merge Modules are the only packages that may have multiple languages. For more information, see Multiple Language Merge Modules. Transform - In a transform file, only one language may be specified. The specified platform and language determine whether a transform can be applied to a particular database. The platform property and the language property can be left blank if no transform restriction relies on them to validate the transform. This summary property is REQUIRED. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
如果这是一个 64 位的 Windows 安装程序，请在模板摘要信息属性中输入 Intel64。请注意，安装包不能同时设置 Intel 和 Intel64 属性。如果当前平台与指定的平台之一不匹配，则安装程序将不会处理该包。不指定平台意味着该包是平台无关的。在模板摘要信息属性的语言 ID 字段中输入 0 或将此字段留空，表示该包是语言中立的。根据它是在源安装程序数据库还是转换中，此属性有不同的变体。源安装程序数据库 - 在源安装程序数据库中只能指定一种语言。合并模块是唯一可以包含多种语言的包。有关更多信息，请参见多语言合并模块。转换 - 在转换文件中，只能指定一种语言。指定的平台和语言决定了是否可以将转换应用于特定数据库。如果没有转换限制依赖于平台属性和语言属性来验证转换，则可以将这些属性留空。此摘要属性是必需的。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty

WordCount Property​ 字数属性

Gets or sets the WordCount summary information property.
获取或设置字数摘要信息属性。

Declaration​ 声明

public int WordCount { get; set; }

Remarks​ 备注

The WordCount summary information property indicates the type of source file image. If this property is not present, it defaults to 0. Note that this property is stored in place of the standard Count property. This property is a bit field. New bits may be added in the future. At present the following bits are available:
WordCount 摘要信息属性指示源文件图像的类型。如果此属性不存在，则默认为 0。请注意，此属性存储在标准 Count 属性的位置。此属性是一个位字段。将来可能会添加新的位。目前可用的位如下：

  • Bit 0: 0 = long file names, 1 = short file names
    位 0：0 = 长文件名，1 = 短文件名
  • Bit 1: 0 = source is uncompressed, 1 = source is compressed
    位 1：0 = 源未压缩，1 = 源已压缩
  • Bit 2: 0 = source is original media, 1 = source is administrative installation
    位 2：0 = 来源是原始媒体，1 = 来源是管理安装
  • [MSI 4.0] Bit 3: 0 = elevated privileges can be required to install, 1 = elevated privileges are not required to install
    [MSI 4.0] 位 3：0 = 可能需要提升权限进行安装，1 = 不需要提升权限进行安装

These are combined to give the WordCount summary information property one of the following values indicating a type of source file image:
这些组合起来为 WordCount 概要信息属性提供以下值之一，指示源文件图像的类型：

  • 0 - Original source using long file names. Matches tree in Directory table.
    0 - 使用长文件名的原始来源。与目录表中的树匹配。
  • 1 - Original source using short file names. Matches tree in Directory table.
    1 - 使用短文件名的原始来源。与目录表中的树匹配。
  • 2 - Compressed source files using long file names. Matches cabinets and files in the Media table.
    2 - 使用长文件名的压缩源文件。与媒体表中的柜和文件匹配。
  • 3 - Compressed source files using short file names. Matches cabinets and files in the Media table.
    3 - 使用短文件名压缩源文件。匹配媒体表中的柜和文件。
  • 4 - Administrative image using long file names. Matches tree in Directory table.
    4 - 使用长文件名的管理映像。匹配目录表中的树。
  • 5 - Administrative image using short file names. Matches tree in Directory table.
    5 - 使用短文件名的管理映像。匹配目录表中的树。

Note that if the package is marked as compressed (bit 1 is set), the installer only installs files located at the root of the source. In this case, even files marked as uncompressed in the File table must be located at the root to be installed. To specify a source image that has both a cabinet file (compressed files) and uncompressed files that match the tree in the Directory table, mark the package as uncompressed by leaving bit 1 unset (value=0) in the WordCount summary information property and set «see F:WixToolset.Dtf.WindowsInstaller.FileAttributes.Compressed» (value=16384) in the Attributes column of the File table for each file in the cabinet. For a patch package, the WordCount summary information property specifies the patch engine that was used to create the patch files. The default value is 1 and indicates that MSPATCH was used to create the patch A value of "2" means that the patch is using smaller, optimized, files available only with Windows Installer version 1.2 or later. A patch with a WordCount of "2" fails immediately if used with a Windows Installer version earlier than 1.2. A patch with a WordCount of "3" fails immediately if used with a Windows Installer version earlier than 2.0. This summary property is REQUIRED. Win32 MSI APIs: MsiSummaryInfoGetProperty , MsiSummaryInfoSetProperty
请注意，如果包被标记为压缩（设置了位 1），安装程序只安装位于源的root的文件。在这种情况下，即使在文件表中标记为未压缩的文件也必须位于root才能安装。要指定一个同时具有内阁文件（压缩文件）和与目录表中的树匹配的未压缩文件的源映像，请通过在 WordCount 摘要信息属性中将位 1 保留为未设置（值=0）来将包标记为未压缩，并在文件表的 Attributes 列中为每个内阁文件设置«see F:WixToolset.Dtf.WindowsInstaller.FileAttributes.Compressed»（值=16384）。对于patch包，WordCount 摘要信息属性指定用于创建patch文件的patch引擎。默认值为 1，表示使用 MSPATCH 创建了patch。值为“2”表示patch使用了仅在 Windows Installer 1.2 或更高版本中可用的更小、优化的文件。WordCount 为“2”的patch如果与早于 1.2 版本的 Windows Installer 一起使用会立即失败。WordCount 为“3”的patch如果与早于 2.0 版本的 Windows Installer 一起使用会立即失败。此摘要属性是必需的。Win32 MSI API：MsiSummaryInfoGetProperty，MsiSummaryInfoSetProperty
